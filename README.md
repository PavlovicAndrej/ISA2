### About program
This is tool for DNS tunneling implementing both client and server.

### Compile 
**make** from root dir

### Run example
**dns_receiver example.com received/**

**dns_sender -u 127.0.0.1 -s 0 example.com receive.txt ./send.txt**

For help run them without parameters.
